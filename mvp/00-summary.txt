WASD MVP - MISSING FEATURES SUMMARY
====================================

WHAT WORKS NOW:
- Full compilation pipeline (source -> LLVM -> binary)
- Parser for all language constructs
- Type inference and checking
- Borrow checker (ownership, borrows)
- Basic IR lowering
- LLVM codegen for primitives
- Code formatter
- Package manager (GitHub releases)
- Basic LSP (diagnostics only)
- Basic REPL (expressions only)

HIGH PRIORITY (COMPLETED for MVP):

1. [x] LOOPS CODEGEN
   - while/for fully implemented with break/continue
   - IR lowering in stmt.rs, LLVM codegen in terminator.rs

2. [x] STDLIB RUNTIME
   - Vec and HashMap implemented in runtime/wasd_runtime.c
   - C FFI for runtime functions

3. [x] TEST RUNNER
   - wasd test command implemented
   - #[test] and #[ignore] attributes working
   - Test filtering with --filter

4. [x] TRAIT ENFORCEMENT
   - Trait registration and impl checking
   - Error on missing trait methods

5. [x] CLOSURE CAPTURE
   - Lambda lifting implementation
   - Captured variables passed as extra parameters

MEDIUM PRIORITY:

6. LSP completeness (hover, goto def)
7. Error context/chaining
8. Defer statement
9. REPL improvements
10. Documentation generator

LOW PRIORITY (v2):

- Comptime execution
- Macros
- Async runtime
- WebAssembly target
- Refinement types
- Full concurrency primitives
